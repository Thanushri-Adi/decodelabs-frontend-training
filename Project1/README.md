# Project 1: Static Webpage Design

## Objective
Master the structural foundations of web design through strict semantic markup and modular styling.

## Key Implementations
* **Information Architecture:** Organized content through a logical layout sitemap before writing code.
* **Semantic Outlining:** Built a strict semantic tree (`<header>`, `<main>`, `<nav>`, `<article>`, `<aside>`, `<footer>`) with a single `<h1>` and zero skipped heading levels.
* **BEM Nomenclature:** Applied the **BEM (Block-Element-Modifier)** naming convention for clean, modular, and DRY (Don't Repeat Yourself) CSS.
* **Low Specificity Curve:** Maintained a low specificity curve by completely avoiding the use of styling IDs.
* **Performance Control:** Prevented Cumulative Layout Shift (CLS) by mapping explicit image aspect ratios directly onto graphic elements.
